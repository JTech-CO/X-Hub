개요

이 문서는 X Reposts Raffle Extension(이하 “확장 프로그램”)의 개인정보 처리방침입니다. 본 확장 프로그램이 사용자 데이터를 처리하는 경우, 본 방침은 정확하고 최신 정보를 제공합니다.

확장 프로그램은 기본적으로 브라우저 내에서 동작하며, 화면에 공개된 리포스트 목록 및 팔로워/팔로잉 목록을 읽어 추첨(Raffle 모드) 및 맞팔로우 확인(Follow 모드)을 수행합니다. 어떠한 사용자 데이터도 외부 서버로 전송하지 않습니다.

1. 사용자 데이터 처리 여부

확장 프로그램은 페이지에서 보이는 공개 정보(사용자 아이디/핸들, 닉네임, 프로필 설명, 팔로우 상태 등)를 일시적으로 처리하여 목록 정렬, 추첨, 맞팔로우 여부 확인 기능을 수행합니다. 또한 사용 편의를 위해 UI 설정(모드 상태, 당첨자 수 등)을 브라우저 로컬 저장소에 보관할 수 있습니다.

2. 포괄적 공개

2-a. 수집·이용·공유 방법

  • 수집: 사용자가 X(트위터) 리포스트 페이지 또는 팔로워/팔로잉 페이지를 열고 기능을 실행하면, 확장 프로그램은 화면의 공개 DOM을 파싱하여 계정 정보를 읽습니다.
  • 이용: 수집된 데이터는 중복 제거, 필터링, 무작위 추첨, 맞팔로우 상태 스캔 및 결과 내보내기(클립보드 복사)에만 사용됩니다.
  • 보관: 추첨 결과와 스캔 목록은 실행 중인 메모리에만 존재하며, 사용자가 탭을 닫거나 브라우저를 초기화하면 즉시 소멸합니다.
  • 공유: 프로그램 내부적으로 데이터를 외부와 자동 공유하지 않습니다. 단, 사용자가 텍스트, CSV, JSON 형태로 직접 '복사(Copy)' 버튼을 눌러 내보내는 경우 그 수신자는 사용자가 결정합니다.

2-b. 공유 대상

기본 기능에서 사용자 데이터를 외부 제3자에게 절대 제공하지 않습니다. 어떠한 분석 도구, 광고 SDK, 원격 서버, 클라우드 전송 기능도 포함되어 있지 않습니다. 사용자가 자발적으로 결과 데이터를 복사하여 외부 플랫폼에 게시하는 경우에만 데이터가 공유됩니다.

  • 아동: 아동을 특별히 대상으로 하지 않는 유틸리티 도구입니다.

Overview

This Privacy Policy explains how X Reposts Raffle Extension (“the Extension”) handles user data. Whenever the Extension processes user data, this policy provides accurate and up‑to‑date information.

The Extension works strictly locally within your browser. It reads the public retweeters list and followers/following lists rendered on the page to run raffles (Raffle mode) and mutual follow checks (Follow mode). Absolutely no data is sent to any external server.

1. Whether We Process User Data

The Extension temporarily processes publicly visible information on the active page (e.g., usernames/handles, display names, bios, follow status) to sort participants, draw winners, and check mutual follow status. For your convenience, basic UI preferences may be stored securely in your browser's local storage.

2. Comprehensive Disclosure

2-a. How We Collect, Use, and Share

  • Collect: When you open an X (Twitter) retweets or followers/following page and start the scan, the Extension parses the public DOM to read account details.
  • Use: The extracted data is strictly used for de-duplication, filtering, random drawing, mutual follow scanning, and exporting results via clipboard.
  • Store: Raffle and scan lists exist only in active memory and vanish as soon as the tab is closed. Settings persist only locally.
  • Share: We do not share any data automatically. If you choose to copy results (Text/CSV/JSON), you are in full control of where that data goes.

2-b. Parties With Whom Data Is Shared

By default, we do not share user data with any third parties. We strictly do not use tracking analytics, ad SDKs, remote servers, or cloud transfers. Data is only shared if you voluntarily copy the exported results and post them elsewhere.

Additional Information

  • Security: All data processing is strictly confined to your local browser environment.
  • Children: This is a general utility tool and is not directed at children.
